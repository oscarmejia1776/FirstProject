Markup is text that has special meaning. By using things called "tags" one can change how text is portrayed on a page.
There are opening and closing tabs. Both of these along with text between them combine to form an element

For instance "em" makes text italic and "strong" makes text bold.

Writing an element using the tag "mark" will highlight the text. Using the tags "sub" and "sup" will be in reference to subscript or superscript.

The only tag we've covered so far that is a "void tag" or "void element" is "br" or line break.

Inline Elements

  1. When making an ordered list you have to start with tag "ol"
  2. Next you have to put "li" in front of each list item, but you don't need to put a closing tag.
  3. You must close the list with the closing tag "/ol"
  • When making an unordered list, it works the same way as an ordered list but you must use the tag "ul"
  • The default for an unordered list is bullets, while ordered lists is numbers.
    1. You can add lists within lists as well. But you mist remember to close the list.
  • The elements so far that have closing tags are "p" and "li". "br" is still a void tag and only needs an opening.

If I write an opening tag of "p" but do not add a closing tag, it will still work because the next tag will signify the beginning a new piece of material.
